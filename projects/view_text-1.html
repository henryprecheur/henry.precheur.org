title: A tiny text viewer: view_text date: 2007-10-16 files: view_text-1.tar.gz

view_text is a tiny text viewer. It can be used to preview text resume.

Its conception was heavily inspired by these articles:

  • Ten Tips for a (Slightly) Less Awful Resume
  • How to Create and E-Mail a Plain Text Resume

It allows you to quickly preview your text resume. You can change the font and strip out the white-spaces. It features an easy scrolling too. view_text was developed and tested under OpenBSD. It should work in every POSIX systems. It needs GTK+2 (at least 2.8) and is written in ISO C99.
