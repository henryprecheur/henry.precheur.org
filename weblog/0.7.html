title: Weblog 0.7 date: 2008-04-07 23:30:42

Here is the 7th release of Weblog. It adds minor enhancements and fixes small problems.

  • More meaningful error messages
  • Added --quiet option
  • Added new date format (it is now possible to specify hours, minutes and seconds in the date)
  • Added a div around the main blog's content
  • Weblog now requires at least Python 2.5

For more informations check the Weblog's page or download Weblog 0.7 directly.
You can also grab the standalone version of Weblog, which includes Jinja: Weblog 0.7 + Jinja.
