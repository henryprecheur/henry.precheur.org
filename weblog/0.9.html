title: Weblog 0.9 released date: 2008-07-21 22:52:57

This is the last version of Weblog before the 1.0.

List of changes:

  • Added support for Jinja2
  • Added directive 'files' to copy files attached to a post
  • Added documentation about visual customization
  • Made Weblog more customizable
  • Many bug-fixes and small improvements

Download Weblog 0.9 or Weblog 0.9 + Jinja2 2.0 (standalone version).

More informations on Weblog's page. If you have any problems with Weblog do not hesitate to ask: henry@precheur.org, I will be glad to help ;)

The next version should not only more documentation, examples and bug fixes.
