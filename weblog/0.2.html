title: Weblog 0.2 date: 2007-8-29

The version 0.2 of Weblog has been released!

Changes include:

  • Added configuration file
  • Weblog is now HTML v4.01 compliant
  • Documentation and examples were added
  • Installation scripts added
  • Fixed some bugs

Weblog is in its early development stage. So dont expect too much of it for now. But if you simply want a quick way of posting HTML articles on the web, it might be a good pick.
