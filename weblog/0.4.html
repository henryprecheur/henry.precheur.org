title: Weblog 0.4 date: 2008-1-24

Another release of Weblog! Weblog has now all its essential features. If you are looking for a simple blog engine, Weblog is the way to go! It does not require any database or server-side configuration. Just generate static files and upload them to your web directory and you are done.

ChangeLog:

  • Command line interface
  • Pagination
  • Limitation of the number of post in the RSS file
  • Added head, header and footer definition in the HTML file. Allow you to have a custom logo / search box for example. Also you can now include a custom CSS without editing the templates.
  • Copy extra files via the extra_files directive
  • Better error handling and meaningful error messages
  • Added initial unit tests
