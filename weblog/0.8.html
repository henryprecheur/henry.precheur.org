title: Weblog 0.8 date: 2008-05-01 22:46:57

Weblog 0.8 is out. Starting from this release the Weblog executable is named: weblog instead of weblog_publish.py. Weblog is now feature complete, the next release will focus on documentation and bug fixes.

Changes:

  • Reworked the command line interface, publish is now a command of the weblog executable
  • Added date command to set post's file date
  • Documentation edition and update (Many kudos to Anis)
  • More unit tests
  • Fixed installation using easy_install
  • Consistent error messages

Download Weblog 0.8 or the standalone version Weblog 0.8 + Jinja 1.2.
