title: Weblog 0.5 date: 2008-2-5

Weblog 0.4 was mostly unusable due to a bug in the renderstring jinja filter. So I decided to quickly release a new version of Weblog to address this bug and a couple of other problems.

ChangeLog:

  • Fixed critical bug when extra_head was not defined
  • Fixed buggy link in post page's title
  • Added more unit tests for files generation
  • Removed helpers.html.tmpl

To learn more: Weblog's page

Or download Weblog-0.5 directly.
